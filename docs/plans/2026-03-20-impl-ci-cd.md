# CI/CD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Automated test runs on push/PR and automated PyPI releases on version tags.

**Architecture:** Two GitHub Actions workflows — `test.yml` runs pytest on every push/PR, `release.yml` builds and publishes on version tags. A version consistency check script prevents mismatched versions across the three files that declare it.

**Tech Stack:** GitHub Actions, `astral-sh/setup-uv`, pytest, uv build/publish

**Spec:** `docs/plans/2026-03-20-ci-cd.md`

**Execution order:** This plan has no dependencies and should be implemented first. Plan 2 (Plugin Distribution) depends on the version check script created here. Note: Plan 2 will later modify the version check script to handle `importlib.metadata` — this is expected.

### Task 1: Version Consistency Check Script

**Files:**
- Create: `scripts/check_version.py`
- Test: Run the script directly — it's self-validating

- [ ] **Step 1: Write the version check script**

```python
#!/usr/bin/env python3
"""Verify that version strings are consistent across the project."""

import json
import re
import sys
from pathlib import Path

ROOT = Path(__file__).resolve().parent.parent

def get_pyproject_version() -> str:
    text = (ROOT / "pyproject.toml").read_text()
    match = re.search(r'^version\s*=\s*"(.+?)"', text, re.MULTILINE)
    if not match:
        sys.exit("Could not find version in pyproject.toml")
    return match.group(1)

def get_init_version() -> str:
    text = (ROOT / "ue_eyes" / "__init__.py").read_text()
    match = re.search(r'^__version__\s*=\s*"(.+?)"', text, re.MULTILINE)
    if not match:
        sys.exit("Could not find __version__ in ue_eyes/__init__.py")
    return match.group(1)

def get_plugin_version() -> str:
    data = json.loads((ROOT / ".claude-plugin" / "plugin.json").read_text())
    return data["version"]

def main() -> None:
    versions = {
        "pyproject.toml": get_pyproject_version(),
        "ue_eyes/__init__.py": get_init_version(),
        ".claude-plugin/plugin.json": get_plugin_version(),
    }
    unique = set(versions.values())
    if len(unique) == 1:
        print(f"All versions match: {unique.pop()}")
    else:
        print("Version mismatch detected:")
        for file, ver in versions.items():
            print(f"  {file}: {ver}")
        sys.exit(1)

if __name__ == "__main__":
    main()
```

- [ ] **Step 2: Run it to verify it passes with current state**

Run: `uv run python scripts/check_version.py`
Expected: `All versions match: 0.1.0`

### Task 2: Test Workflow

**Files:**
- Create: `.github/workflows/test.yml`

- [ ] **Step 1: Write the test workflow**

```yaml
name: Tests

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    timeout-minutes: 5
    steps:
      - uses: actions/checkout@v4

      - uses: astral-sh/setup-uv@v4
        with:
          version: "0.6.x"

      - name: Install dependencies
        run: uv sync

      - name: Check version consistency
        run: uv run python scripts/check_version.py

      - name: Run tests
        run: uv run pytest tests/ -v
```

### Task 3: Release Workflow

**Files:**
- Create: `.github/workflows/release.yml`

- [ ] **Step 1: Write the release workflow**

```yaml
name: Release

on:
  push:
    tags: ["v*"]

jobs:
  release:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    permissions:
      contents: write
    steps:
      - uses: actions/checkout@v4

      - uses: astral-sh/setup-uv@v4
        with:
          version: "0.6.x"

      - name: Install dependencies
        run: uv sync

      - name: Check version consistency
        run: uv run python scripts/check_version.py

      - name: Run tests
        run: uv run pytest tests/ -v

      - name: Build package
        run: uv build

      - name: Publish to PyPI
        run: uv publish
        env:
          UV_PUBLISH_TOKEN: ${{ secrets.PYPI_TOKEN }}

      - name: Create GitHub Release
        uses: softprops/action-gh-release@v2
        with:
          generate_release_notes: true
```

- [ ] **Step 2: Commit**

```bash
git add .github/workflows/release.yml
```
